# Cra3ture — Personal Portfolio

Personal portfolio website for Rey (Cra3ture) — UI/UX designer, motion designer, and animator based in Sri Lanka.

**Live site:** [https://my-repo-self-seven.vercel.app/]((https://my-repo-self-seven.vercel.app/))

## Stack

Plain HTML · CSS · Vanilla JS — no build tools, no dependencies.

## Features

- Light / dark mode (toggle by clicking the logo)
- Idle floating animation on project cards
- Enquire modal with contact form
- CV download prompt
- Fully responsive

## Structure

```
/
├── index.html
├── style.css
├── script.js
└── assets/
    ├── Cra3ture Logo.png   # Dark logo (used in light mode)
    ├── LogoWhite.png        # White logo (used in dark mode)
    ├── Lite.png             # Light mode background
    ├── Dark.png             # Dark mode background
    └── Rey_CV.pdf           # CV (add before deploying)
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Save — your site will be live at `https://<username>.github.io/<repo-name>/`
